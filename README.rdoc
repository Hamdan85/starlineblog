== README

Para rodar o projeto basta installar o rails: http://installrails.com/

Após, entrar no diretorio e rodar o comando 'bundle install' para instalar as gemas necessárias ao projeto.

para migrar a db 'rake db:setup'

e finalmente para iniciar o servidor 'rails s'

ou acessar o aplicativo instalado no heroku: 'http://starlineblog.herokuapp.com'


Please feel free to use a different markup language if you do not plan to run
<tt>rake doc:app</tt>.
